## üñ•Ô∏è Screenshot Instructions

### Method 1: Using Browser (Recommended)
1. Make sure both servers are running:
   - Backend: `cd server && npm run dev`
   - Frontend: `cd client && npm run dev`
2. Open http://localhost:5173 in your browser
3. Navigate to each page
4. Press `F12` to open DevTools
5. Press `Ctrl + Shift + P` and type "Capture full size screenshot"
6. Save each screenshot with the naming convention above

### Method 2: Using Windows Snipping Tool
1. Press `Windows Key + Shift + S`
2. Select the area to capture
3. Save the screenshot to this folder

### Method 3: Using Full Page Screenshot Extensions
- **Chrome/Edge**: Install "Full Page Screen Capture" extension
- **Firefox**: Right-click ‚Üí Take Screenshot ‚Üí Save full page
